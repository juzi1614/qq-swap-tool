# QQ 聊天记录视角互换工具 v3.1

通过修改本地 QQ 数据库实现聊天记录视角互换——把 A 和 B 的对话，变成看起来像是 C 和 B 的对话。

## 原理

QQ 的聊天记录存储在本地加密 SQLite 数据库中。本工具解密数据库后，将消息的发送者 UID 替换，实现视角互换。

## 快速开始

### 方式一：免安装版（推荐）

下载 [QQ-Swap-Tool.exe](../../releases) 直接运行，无需安装 Python。

### 方式二：源码运行

```bash
pip install sqlcipher3
python gui.py
```

## 使用步骤

1. 关闭 QQ
2. 获取密钥（GUI 中点"获取"，或手动运行 `windows_ntqq_get_key.ps1`）
3. 填写账号参数并点击"开始互换"

## 参数说明

| 参数 | 说明 |
|------|------|
| 源账号 | 聊天记录提供方 |
| 目标对话 | 要互换的对话对象 |
| 输出账号 | 部署到哪个 QQ 号 |
| 显示区域 | 聊天显示在哪个对话区域 |
| 右侧显示 | 谁显示在右侧（"我"） |

## 注意事项

- 输出账号的目标 peer 必须无服务器聊天记录，否则只显示最近几条
- 操作前会自动备份数据库
- 仅供学习研究，使用者自行承担风险

## 依赖

- Python 3.10+
- sqlcipher3
